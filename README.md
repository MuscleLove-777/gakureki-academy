# 学歴アカデミー

偏差値だけでは見えない、大学の本当の価値を可視化するサイト。

## 機能

- **偏差値ランキング**: 主要49大学を偏差値・就職力でソート・フィルター・検索
- **大学群一覧・詳細**: 東京一工/旧帝大/早慶上理ICU/MARCH/関関同立/成成明学/日東駒専/産近甲龍/大東亜帝国
- **就職・採用データ**: 業界別の学歴フィルター強度、偏差値帯別の狙える業界マトリクス
- **大学群比較ツール**: 2つの大学群を選んで偏差値・就職力・年収を比較
- **学歴フィルター診断**: 偏差値を入力して狙える業界を診断

## 技術構成

- HTML / CSS / JavaScript（フレームワーク不使用）
- レスポンシブ対応（スマホ・PC）
- GitHub Pages でホスティング
